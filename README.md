# Unity_KeyEventManager

<img src="https://github.com/XJINE/Unity_KeyEventManager/blob/main/Screenshot.png" width="100%" height="auto" />

KeyEventManager provides a key-based event handling system.

## Importing

You can use Package Manager or import it directly.

```
https://github.com/XJINE/Unity_KeyEventManager.git?path=Assets/Packages/KeyEventManager
```

### Dependencies

This project use following resources.

- Unity.InputSystem

## How to Use

Register your key and function pairs in the Inspector.
